Text Adventure

Sobre el proyecto

Este proyecto es una aventura de texto interactiva, desarrollada con Java y la biblioteca de interfaces gráficas Swing.

La aplicación permite al jugador explorar una narrativa básica mediante la selección de opciones que afectan el curso de la historia.

Tecnologías

  • Java
  • Swing
  • Maven

Estructura

El proyecto sigue una arquitectura de tres capas para mantener la separación de preocupaciones y facilitar el mantenimiento del código.

  • ui: Interfaz gráfica del Usuario. Clases de vista con sus respectivos controladores.
  • logic: Gestiona las reglas y lógica del juego.
  • data: Clases de datos del juego

Créditos

Diseño de la interfaz gráfica e historia inspirada en la del tutorial de RyiSnow (YouTube)
